# Dashboard Data

Dashboard Data is a repository that consolidates publicly available datasets related to Universiti Teknologi MARA. This repository serves as a centralised resource webmaster to access organised, reliable, and up-to-date information about the universityâ€™s campuses, faculties, academic programs, statistics, and other non-confidential institutional data.

It supports data-driven initiatives, promotes transparency, and enables the webmaster to build applications, dashboards and analyses.
